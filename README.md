# CodeAlpha_Email-Extractor
# 📧 Email Extractor using Python

## 📌 Project Overview

This project is developed as **Task 2** for the **CodeAlpha Python Programming Internship**.

The Email Extractor is a simple Python application that scans a text file, identifies all valid email addresses using **Regular Expressions (Regex)**, and saves the extracted emails into a separate output file.

---

## 🚀 Features

- 📂 Reads data from a text file
- 📧 Extracts valid email addresses
- 🔍 Uses Python Regular Expressions (Regex)
- 💾 Saves extracted emails to an output file
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- Python 3
- Regular Expressions (`re` module)
- File Handling

---

## 📁 Project Structure

```
Email-Extractor/
│
├── input.txt          # Input text containing emails
├── output.txt         # Extracted email addresses
├── email_extractor.py # Main Python program
└── README.md
```

---

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/yourusername/Email-Extractor.git
```

2. Navigate to the project folder

```bash
cd Email-Extractor
```

3. Add your content with email addresses to `input.txt`

4. Run the program

```bash
python email_extractor.py
```

5. The extracted email addresses will be saved in `output.txt`.

---

## 📷 Example

### Input (`input.txt`)

```
Contact us:
support@example.com
info@company.org
hello123@gmail.com
```

### Output (`output.txt`)

```
support@example.com
info@company.org
hello123@gmail.com
```
